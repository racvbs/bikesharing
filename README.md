# bikesharing

Module 14 - Challenge

## Purpose

To understand NYC trips on bikes with the analysis of data in Tableau.

## Information

We have data of trips on bike in the next database columns:

* Trip duration
* Start time
* Stop time
* Start station:
*   Id
*   Name
*   Latitude
*   Longitude
* Stop station:
*   Id
*   Name
*   Latitude
*   Longitude
* Bike id
* User type 
* Birth year
* Gender (if they want to answer)
